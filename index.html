Reference Intervals

Establishment, Validation & Transfer

For Biochemical Quantitative Projects

Based on CLSI EP28-A3c Guidelines

Overview

Reference intervals are essential tools in laboratory medicine

They define the expected range of test results in healthy individuals

Proper establishment ensures accurate clinical interpretation

Today we'll explore a systematic approach to reference interval establishment

Key Concepts

Reference Individual

A person selected using well-defined criteria, representing the target population

Reference Population

The entire set of individuals who meet reference individual criteria

Reference Value

A test result obtained from a reference individual

Reference Interval

The interval between and including two reference limits (typically central 95%)

Reference Interval vs. Clinical Decision Limit

Reference Interval

Based on healthy reference population

Methods for Establishing Reference Intervals

A Priori Method

  • Direct sampling approach
  • Pre-defined selection criteria
  • Minimum 120 reference individuals per partition
  • Gold standard methodology

A Posteriori Method

  • Indirect sampling approach
  • Uses existing laboratory data
  • Statistical methods to identify healthy population
  • Useful when direct sampling is challenging

A Priori Method: The Process

1

Define Selection Criteria

Age, sex, ethnicity, health status, medication use

2

Recruit Reference Individuals

Screen potential participants using questionnaires and basic tests

3

Collect and Process Samples

Follow standardized pre-analytical protocols

4

Analyze and Establish Intervals

Statistical methods to determine central 95% interval

CLSI EP28-A3c recommends minimum sample size of 120 individuals per partition

Statistical Methods

Non-parametric Method

Rank-based approach using percentiles

Does not assume normal distribution

Recommended for n ≥ 120

Parametric Method

Assumes Gaussian distribution

Uses mean ± 1.96 SD

May require data transformation

Robust Method

Less sensitive to outliers

Useful for smaller sample sizes

Iterative approach to estimate central tendency

Handling Outliers

Dixon Method

For sample sizes between 3 and 25

Compare extreme value to next closest value

D/R ratio compared to critical value

Tukey Method

Uses interquartile range (IQR)

Outliers: < Q1-1.5×IQR or > Q3+1.5×IQR

Effective for larger datasets

Removal of outliers should be done cautiously and with statistical justification

Partitioning Reference Intervals

Consider partitioning when biologically relevant differences exist

Common Partition Factors

  • Sex
  • Age
  • Ethnicity
  • Pregnancy
  • Fasting status

Statistical Criteria

  • Harris & Boyd method
  • Lahti criterion
  • z-statistic evaluation
  • Non-overlapping 90% CI of reference limits

Example: Creatinine typically requires sex-partitioned reference intervals

Males: 0.7-1.2 mg/dL
Females: 0.5-1.0 mg/dL

Reference Interval Transfer

De novo establishment is resource-intensive

CLSI EP28-A3c provides framework for transferring reference intervals

Conditions for Transfer

  • Comparable reference populations
  • Similar pre-analytical conditions
  • Analytical systems with demonstrated commutability
  • Successful validation in target setting

Validation Process

1

Select 20 Reference Individuals

Same criteria as original reference population

2

Test Samples

Use the same analytical methodology

3

Evaluate Results

Apply the 10% rule for acceptance

If ≤ 2 out of 20 results fall outside RI: Accept RI

If 3-4 out of 20 results fall outside RI: Collect 20 more samples

If > 4 out of 20 results fall outside RI: Reject RI

If ≤ 4 out of 40 results fall outside RI (after additional collection): Accept RI

Key Requirements

  • Standardized pre-analytical procedures
  • Harmonized analytical methods
  • Uniform selection criteria
  • Centralized data analysis
  • Rigorous quality control measures

Example: The CALIPER project established pediatric reference intervals across multiple centers in Canada

Pre-analytical Considerations

Pre-analytical factors significantly impact reference interval studies

Subject Preparation

  • Fasting status
  • Physical activity
  • Posture
  • Diurnal variation
  • Medication use

Sample Collection

  • Collection site
  • Collection tube type
  • Tourniquet time
  • Order of draw
  • Collection technique

Sample Handling

  • Transport conditions
  • Processing time
  • Centrifugation parameters
  • Storage temperature
  • Freeze-thaw cycles

Analytical Quality Requirements

Method Validation

Verify accuracy, precision, linearity, and analytical specificity

Traceability

Results should be traceable to recognized reference materials or methods

Internal Quality Control

Consistent monitoring throughout the study period

External Quality Assessment

Participation in proficiency testing programs

Analytical performance directly impacts the width of reference intervals

RI width = Biological variation + Analytical variation

Documentation and Periodic Review

Essential Documentation

  • Selection criteria for reference individuals
  • Demographic characteristics of the reference population
  • Pre-analytical procedures
  • Analytical methods and performance
  • Statistical methods and raw data
  • Partitioning decisions with justification

Case Study 1: Multicenter Study

CLSI-Derived Reference Intervals for Eastern and Southern Africa

Background

Established reference intervals for 25 hematology, immunology, and biochemistry parameters

Methods

  • CLSI guidelines followed
  • 2,990 healthy adults from Uganda, Kenya, Zambia
  • Standardized pre-analytical and analytical procedures

Results

Significant differences from US-derived intervals for multiple analytes

Regional differences necessitated partitioning by geographical location

Case Study 2: Age and Sex Partitioning

CALIPER Initiative: Pediatric Reference Intervals

Background

Comprehensive pediatric reference intervals across all age groups

Methods

  • Recruited over 8,500 healthy children and adolescents
  • Statistically assessed need for age and sex partitioning
  • Used CLSI C28-A3 guidelines

Results

Many analytes required multiple age partitions

Sex differences became significant at puberty for many parameters

Established dynamic biological profiles across pediatric development

Challenges and Future Directions

Current Challenges

  • Resource constraints for de novo establishment
  • Defining "health" in reference populations
  • Harmonization across laboratories
  • Addressing gaps in pediatric and geriatric ranges
  • Accounting for genetic and ethnic variability

Future Directions

  • Big data approaches using indirect methods
  • AI/ML for outlier detection and pattern recognition
  • Global harmonization initiatives
  • Integration of -omics data
  • Personalized reference intervals

Key Takeaways

  • Reference intervals are fundamental tools for result interpretation
  • Establishment should follow standardized approaches (CLSI EP28-A3c)
  • Sample size, statistical methods, and partitioning require careful consideration
  • Transfer and validation offer practical alternatives to de novo establishment
  • Quality assurance across pre-analytical and analytical phases is critical
  • Documentation and periodic review ensure continued applicability

Proper reference intervals are essential for accurate clinical decision-making
